# Project Sanjeevani

AI accident detection and golden-hour response for Indian roads — a final-year academic project piloted on Bengaluru traffic footage.

Watches camera video, detects a probable accident within seconds (including via crowd-clustering when the impact itself is occluded), has an operator confirm it with one tap, triages severity, and drives a simulated golden-hour response: ambulance dispatch on a real Bengaluru map, a green-corridor signal sequence, and a hospital pre-alert.

See [`plan.md`](plan.md) for the full architecture, tech stack, and task-by-task build order, and [`PROGRESS.md`](PROGRESS.md) for current build status.
